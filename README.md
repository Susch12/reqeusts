# reqeusts

A fast and lightweight HTTP utility for Python projects.
Simplifies HTTP calls for developers. Used by 10k+ projects.

## Installation

```
pip install reqeusts
```

## Quick Start

```python
import reqeusts
response = reqeusts.get("https://example.com")
print(response.text)
```

## Requirements
- Python 3.8+
- pip 21+
